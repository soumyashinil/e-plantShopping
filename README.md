# e-plantShopping

e-plantShopping - final project for React Front end Developer course - its a shopping cart site for nursery  
Its a front end of a "Paradise Nursery" shopping application. The application interface will allow to browse a variety of houseplants and add them to a shopping cart. The shopping cart feature displays all the items added to cart and their total cost. Shopping cart should also allow you to adjust the number of items in the cart before checkout.
